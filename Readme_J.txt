======================================================================
 XKeymacs - Windows 用 キーボードユーティリティ
======================================================================

--- 概要 ---
XKeymacs は Windows 上の全てのアプリケーションにおいて
emacs like な操作性を提供します。
各機能のオン/オフは、アプリケーション毎に設定できます。


--- インストール ---
1. xkeymacs???.zip を下記サイト等から取得 (??? はバージョン番号)
http://www.cam.hi-ho.ne.jp/oishi/
2. xkeymacs???.zip を解凍
-> xkeymacs??? というディレクトリが得られる。
3. xkeymacs??? ディレクトリを適当な場所へ移動
4. xkeymacs??? ディレクトリの中の xkeymacs.exe を起動


--- アンインストール ---
1. XKeymacs を終了 (終了方法は後述)
2. xkeymacs??? ディレクトリを削除
3. レジストリエディタで、次の値を削除
HKEY_CURRENT_USER\Software\oishi\XKeymacs2
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run\XKeymacs


--- 使用方法 ---
起動方法:
xkeymacs.exe を実行する。
XKeymacs が起動すると、タスクバーに金の鍵が表示されます。

終了方法:
タスクバーの金(もしかしたら銀かも)の鍵のアイコンを右クリックし、
開いたメニューから "終了(X)" を選択する。

各種設定方法:
金の鍵のアイコンを右クリックすると、XKeymacs のメニューにアクセスできる。

プロパティ...
アプリケーション毎に、XKeymacs の各機能を有効にするかどうか設定できる。
特に設定しなければ、標準の設定が利用される。
ダイアログには、ダイアログ用の設定を利用できる。
プロパティダイアログには、Basic、Advanced、List という 3 つのタブがある。
全て同じデータを異なる形式で表示する。
Basic タブではチェックボックスを用い、簡単に Emacs の様な設定を行える。
Advanced タブでは、任意のコマンドに任意のキーを割り当てることができる。
List タブでは、行った設定内容を一覧できる。

設定の取り込み...
「設定の書き出し」によって書き出されたファイルを読み込む。

設定の書き出し...
HKEY_CURRENT_USER\Software\Oishi\XKeymacs2 以下のデータを
ファイルに書き出す。
プロパティダイアログ、および、カスタマイズダイアログで
設定した情報をファイルに書き出す。
主に、他のマシンに設定を移動する際に利用する。

オプション...
Windows にログオンしたときに自動的に XKeymacs を起動させたり、
コマンドプロンプト(MS-DOS プロンプトではない)でタブキーを押したときに
ファイル名、及び、ディレクトリ名を補完させたり、
タスクバーのステータス領域に C-x, Meta, Shift, Ctrl, Alt の
状態を表示させたりできる。

キーボードレイアウト... (Windows NT, 2000, XP, Server 2003, Vista のみ)
キーボードのレイアウトを変更できる。
上のベースキーボードのキーをドラッグ＆ドロップして変更すると、
下のリマップキーボードに赤字で反映される。
変更を取り消すときは、リマップキーボードの赤字のキーをクリック。
Windows が用意しているレジストリを書き換えるだけなので、
XKeymacs を無効にしてもアンインストールしても設定は解除されない。
XKeymacs を用いずに設定を解除するには、
レジストリエディタで以下の値を消せば良い。
HKEY_CURRENT_USER\Keyboard Layout\Scancode Map
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Keyboard Layout\Scancode Map

無効/有効にする
C-q で有効/無効を切り替えることもできる。

リセット
なにかのはずみで XKeymacs が効かなくなったときは、リセットすると直る。

終了
XKeymacs を終了する。
